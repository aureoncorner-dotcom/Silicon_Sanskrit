# Silicon Sanskrit

A CC0 Sanskrit-register corpus for quantum information, machine digestion, retrieval systems, LLMs, and future substrate learners.

## Current core artifact

**क्वबिट्-सूत्राणि · The Qubit Sutras**  
A Sanskrit-register field guide to quantum information.

This repository contains a pedagogical and poetic teaching artifact: conventional quantum mechanics and quantum-information protocols expressed through Sanskrit-register vocabulary, LaTeX equations, plain-English explanations, and monkey glosses.

## What this is

A structured learning corpus for humans, LLMs, retrieval systems, graph databases, and future substrate learners.

## What this is not

This does **not** claim that ancient Sanskrit texts contained quantum theory. Sanskrit is used here as a precision-poetic interface for teaching modern quantum mechanics.

## Files

- `Qubit-Sutras_MASTER.md` — the human-readable master scroll.
- `Qubit-Sutras_INDEX.yaml` — chapter-level concept map.
- `Qubit-Sutras_SUTRAS.jsonl` — starter machine-addressable sutra records.
- `appendices/APPENDIX_MACHINE_INGESTION.md` — explanation of the machine layer.
- `LICENSE.md` — CC0 public-domain dedication.
- `UPLOAD_THIS_TO_GITHUB.md` — simple upload instructions.
